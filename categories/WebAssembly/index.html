Webassembly load dynamic libraries

Ways to load dynamic libraries

There are two types of shared modules:

  1. Main modules, which have system libraries linked in.
  2. Side modules, which do not have system libraries linked in.

1.Set dynamicLibraries

init Module with {dynamicLibraries: [‘A.wasm’,’B.wasm’]} , side modules will be loaded

2 Use LoadDynamicLibraries function

Use Module function “loadDynamicLibrary” to load side module when you need. You need to add “loadDynamicLibrary” in main module’s EXTRA_EXPORTED_RUNTIME_METHODS settings to export this function.
