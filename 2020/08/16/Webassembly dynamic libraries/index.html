Webassembly dynamic libraries main module & side module

Main modoule & Side module

There are two types of shared modules:

  1. Main modules, which have system libraries linked in.
  2. Side modules, which do not have system libraries linked in.

Side modules are created by including the SIDE_MODULE flag when compiler with Emscripten., will not generate the JavaScript file or include any system library functions in the module.

Main modules are created using the MAIN_MODULE flag. This flag tells the Emscripten compiler to include system libraries for dynamic linking.
